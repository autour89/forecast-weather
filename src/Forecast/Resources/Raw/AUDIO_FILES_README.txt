Audio Files Needed
==================

Please add the following audio files to this directory:
1. success.mp3 - Plays when weather data is successfully fetched
2. failure.mp3 - Plays when weather data fetch fails

You can use any short sound effects (1-2 seconds recommended).

Free audio resources:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

The AudioService in Services/AudioService.cs will automatically play these files.
